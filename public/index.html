Geepivo - Pivotal Tracker integration for Google Apps

Pivotal Tracker™ integration for Google Apps™. Provides a gadget for Gmail which allows you to create Stories directly from emails.

This is an Alpha version. It provides only very basic functionality and the UI is rought at the edges. The app is published no mainly to get feedback about which features should be implemented.

  • The gadget talks directly to Pivotal Tracker API. No extra server involved.
  • No settings stored outside your Google Apps account.
  • Requires minimal access rights to your information - only to email subject line and message id.
